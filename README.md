# Using LiquidJS with Express.js

## Get Started

```bash
cd frontend
npm install
cd backend
npm install
```

## Running web application

Start back-end ngrok forwarding to localhost
```bash
cd backend
```
```bash
npm start
```
Start front-end ngrok forwarding to localhost
```bash
cd frontend
```
```
npm start
```

Start ngrok forwarding to localhost
```bash
ngrok http 8080
```
